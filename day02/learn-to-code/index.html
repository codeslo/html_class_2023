Learn to Code

HTML

HTML - Hyper Text Markup Language - is used to add content to a webpage. HTML5 is the latest version.

CSS

CSS - Cascading Style Sheets - is used to control the appearance of a webpage. CSS3 is the latest version.

JavaScript

JavaScript is the programming language of the web. It is used to make webpages dynamic and interactive. The standard JavaScript is based on is called ECMAScript, and a new version is released on a roughly annual basis.
